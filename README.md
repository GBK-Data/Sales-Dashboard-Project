# SQL Queries
## 1. Total Revenue by Region and Country
This helps identify which Regions and Countries generates the most sales and highlights where the business performs the best.
```sql
SELECT Sales.Region, Sales.Country, SUM(Sales.Revenue) AS TotalRevenue
FROM Sales
LEFT JOIN Countries
ON Sales.Country = Countries.CountryName
GROUP BY Sales.Region, Sales.Country
ORDER BY TotalRevenue DESC;
```

## 2. Average Unit Price per Product
Calculates the average price per unit for each product.
```sql
SELECT Sales.Product, Sales.Category, AVG(Sales.UnitPrice) AS AverageUnitPrice
FROM Sales
LEFT JOIN Categories
ON Sales.Category = Categories.CategoryName
GROUP BY Sales.Product, Sales.Category
ORDER BY AverageUnitPrice DESC;
```
